# EggAim
Back-ported Egg Aim move from Banjo Tooie, allowing you to fire eggs in first person. (Press Z to fire)

Also includes an option to fix a vanilla issue where shooting while crouching causes the character always shoot slightly left.

## Changelog
- 0.0.1
    - Initial release
- 0.0.2
    - Added config option to fix crouch shooting left issue. (This is always fixed for first person shooting)
    - Fixed eggs not being able to be shot downwards
