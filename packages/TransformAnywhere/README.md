# TransformAnywhere
Transform into Mumbo's forms anywhere

- D-pad Up: Wishwashy
- D-pad Down: Croc (or Walrus)
- D-pad Left: Bee
- D-pad Right: Termite (or Pumpkin)

D-pad Up as any form returns you to Banjo. Optionally, you can require L to be held while pressing D-pad directions by enabling "Require L to Transform" in the mod settings.
